Vibe Coding 雷达

Vibe Coding 雷达是给 Coding 新手看的项目推荐榜单，整理 90 个最好玩、最好用、最好搓的项目， 并提供 GitHub 来源、MVP 思路、演示效果和一键开工提示词。

国内访问地址：radar.lyihub.com，适合不需要科技上网的朋友直接打开。

  • 最好玩的项目：适合快速做出有互动、有演示效果的作品。
  • 最好用的项目：适合直接进入日常工作流，解决真实问题。
  • 最好搓的项目：适合用 ESP32、Raspberry Pi 等硬件跑通实体反馈。
  • 增长最快的 GitHub 项目：追踪本周正在冒头的开源项目。
  • 必装 Skill：帮新手给 AI 装上手和脚，带着工具一起开工。
