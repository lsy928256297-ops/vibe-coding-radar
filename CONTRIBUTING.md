# Contributing

欢迎给 Vibe Coding 雷达贡献项目、Skill、链接修正和页面改进。

## 推荐项目时请提供

- 项目名称
- GitHub 或官方来源链接
- Demo / 文档链接，如果有
- 它适合新手的原因
- 它属于哪一类：好玩、好用、好搓、增长项目、必装 Skill
- 可能的卡点：API Key、Docker、数据库、模型、硬件、浏览器权限、网络等

## 推荐 Skill / 工具时请提供

- Skill 或工具名称
- 来源链接
- 适合什么项目使用
- 对新手的直接价值
- Star、安装量、官方来源或社区口碑证据，如果有

## 内容原则

- 优先面向 Vibe Coding 新手。
- 优先推荐能看懂、能复现、能尽快跑出 demo 的项目。
- 不把专业垂类工具强行放到新手必装榜。
- 不推荐来源不明、链接失效、演示不可达的项目。
- 推荐 Skill 时要说明它和项目的具体关系，不因为热门就硬塞。
