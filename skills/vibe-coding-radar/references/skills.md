# Skill / Tool Recommendation Reference

Recommend Skills only when they help the project actually start or improve.

## Default beginner tools

### GitHub CLI

- Source: https://cli.github.com/
- Use for: cloning, reading issues/releases, checking repos, opening PRs.
- Recommend when: the project has a GitHub source link.

### OpenAI Skills Catalog

- Source: https://github.com/openai/skills
- Use for: understanding Codex Skill installation and official patterns.
- Recommend when: the user asks what Skills to install or how to extend Codex.

### Playwright Skill

- Source: https://github.com/lackeyjb/playwright-skill
- Use for: browser verification, button/form checks, screenshots, responsive checks.
- Recommend when: the project is a website, web app, game, editor, or dashboard.

### Vercel Agent Skills

- Source: https://github.com/vercel-labs/agent-skills
- Use for: deployment, preview links, web app quality checks.
- Recommend when: the project should become a public link.

## Design and content

### shadcn/ui Skill

- Source: https://github.com/shadcn-ui/ui/blob/main/skills/shadcn/SKILL.md
- Use for: web app components, forms, dialogs, tables, settings panels.
- Recommend when: the project is a SaaS-like tool or dashboard.

### Figma / Canva Skills

- Source: https://claude.com/skills
- Use for: design assets, social graphics, UI context, posters.
- Recommend when: visual output matters.

### 归藏 PPT Skill

- Source: https://github.com/op7418/guizang-ppt-skill
- Use for: Chinese presentation pages, launch decks, visual storytelling.
- Recommend when: the output is a shareable presentation or visual explainer.

## Documents and workflows

### Document Skills

- Source: https://github.com/anthropics/skills
- Use for: PDF, Word, Excel, PowerPoint, OCR, document conversion.
- Recommend when: the project handles files, reports, tables, or documents.

### OpenCLI

- Source: https://github.com/jackwener/OpenCLI
- Use for: browser / website / platform automation.
- Recommend when: the project needs real web operations, data collection, or Chinese platform workflows.

### Hugging Face Skills

- Source: https://github.com/huggingface/skills
- Use for: models, datasets, Gradio, Spaces demos.
- Recommend when: the project needs ML model demos or dataset exploration.
