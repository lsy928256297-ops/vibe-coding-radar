# Project Recommendation Reference

Use this as a compact seed list. Prefer the live Vibe Coding Radar site for browsing, but these examples define the recommendation style.

## 好玩的项目

### AI 小镇 / NPC 社交游戏

- Source: https://github.com/a16z-infra/ai-town
- Demo / official page: https://www.convex.dev/ai-town
- Why: multi-agent social simulation is easy to explain and fun to show.
- Good for: users who want a visible AI demo.
- Watch out: API keys, backend setup, and app configuration.
- Useful Skills / tools: GitHub CLI, OpenAI docs / model examples, Playwright browser checks.

### 手势控制小游戏 / 手势乐器

- Source: https://github.com/collidingScopes/threejs-handtracking-101
- Demo: https://collidingscopes.github.io/threejs-handtracking-101/
- Why: camera + hand tracking gives immediate visual feedback.
- Good for: users who want a playful browser demo.
- Watch out: browser camera permission and device compatibility.
- Useful Skills / tools: GitHub CLI, Playwright browser checks, frontend design.

### 生成式海报 / 壁纸工厂

- Demo: https://multitude-amsterdam.github.io/p5Catalyst/
- Why: visual output is easy to share.
- Good for: posters, wallpapers, social images.
- Watch out: design polish matters more than complex architecture.
- Useful Skills / tools: Canva, Figma, frontend design.

## 好用的项目

### 自动化工作流中枢

- Source: https://github.com/n8n-io/n8n
- Why: turns repeated online tasks into workflows.
- Good for: productivity, weekly reports, data movement, personal automations.
- Watch out: Docker, credentials, webhook setup.
- Useful Skills / tools: OpenCLI, GitHub CLI, web app UI tools.

### PDF 万能工具箱

- Source: https://github.com/Stirling-Tools/Stirling-PDF
- Why: PDF editing is immediately useful.
- Good for: document workflows and internal tools.
- Watch out: local service setup and file handling safety.
- Useful Skills / tools: document Skills, GitHub CLI, browser checks.

### 私人知识问答库

- Source: https://github.com/topoteretes/cognee
- Why: lets users turn their own files into searchable memory / Q&A.
- Good for: personal knowledge bases and document search.
- Watch out: model choice, embeddings, storage, and import formats.
- Useful Skills / tools: OpenAI docs, Hugging Face tools, document Skills.

## 好搓的硬件项目

### ESP32 电子墨水日历牌

- Source: https://github.com/AppForce1/lilygo-t5-47-plus-esphome
- Why: small hardware, visible result, useful at home.
- Good for: first hardware display project.
- Watch out: board model, flashing firmware, Home Assistant / ESPHome setup.
- Useful Skills / tools: GitHub CLI, OpenCLI, hardware docs.

### 智能植物监测器

- Source: https://github.com/agners/epaper-weather-station
- Why: sensor data + display feedback is easy to understand.
- Good for: home hardware demos.
- Watch out: sensor wiring, calibration, and firmware flashing.
- Useful Skills / tools: GitHub CLI, OpenCLI.

### QMK 宏键盘 / 工作流控制台

- Source: https://github.com/qmk/qmk_firmware
- Why: turns keyboard shortcuts into a physical workflow controller.
- Good for: productivity hardware and creator desks.
- Watch out: keyboard compatibility and firmware flashing.
- Useful Skills / tools: GitHub CLI, OpenCLI.
